addTag()method
public function addTag(pid:int, tag_uid:int, tag_text:String, x:Number, y:Number, tags:String):void

Adds a tag with the given information to a photo. See photo uploads for a description of the upload workflow.

Parameters
pid:int — The photo id of the photo to be tagged.
 
tag_uid:int — The user id of the user being tagged. Either tag_uid or tag_text must be specified.
 
tag_text:String — Some text identifying the person being tagged. Either tag_uid or tag_text must be specified. This parameter is ignored if tag_uid is specified.
 
x:Number — The horizontal position of the tag, as a percentage from 0 to 100, from the left of the photo.
 
y:Number — The vertical position of the tag, as a percentage from 0 to 100, from the top of the photo.
 
tags:String — A JSON-serialized array representing a list of tags to be added to the photo. If the tags parameter is specified, the x, y, tag_uid, and tag_text parameters are ignored. Each tag in the list must specify: "x", "y", and either the user id "tag_uid" or free-form "tag_text" identifying the person being tagged. An example of this is the string [{"x":"30.0","y":"30.0","tag_uid":1234567890}, {"x":"70.0","y":"70.0","tag_text":"some person"}].

See also

http://developers.facebook.com/documentation.php?v=1.0&doc=photos

get()method 
public function get(subj_id:int = -1, aid:int = -1, pids:Array = null):void

Returns all visible photos according to the filters specified. This may be used to find all photos of a user, or to query specific eids. Note: It is an error to omit all three of the subj_id, aid, and pids parameters. They have no defaults.

Parameters
subj_id:int (default = -1) — (Optional) Filter by photos associated tagged with this user.
 
aid:int (default = -1) — (Optional) Filter by photos in this album.
 
pids:Array (default = null) — (Optional) Filter by photos in this list. This is a comma-separated list of pids.

See also

http://developers.facebook.com/documentation.php?v=1.0&doc=photos

upload()method 
public function upload(data:String, aid:int = -1, caption:String = ""):void

Uploads a photo owned by the current session user and returns the new photo. See photo uploads for a description of the upload workflow. The only storable values returned from this call are pid, aid, and owner. Note: This signature is different from the facebook api because in Actionscript required parameters are not permitted fteroptional parameters.

Parameters
data:String — (Optional) The raw image data for the photo.
 
aid:int (default = -1) — (Optional) The album id of the destination album.
 
caption:String (default = "") — (Optional) The caption of the photo.
