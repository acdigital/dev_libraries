send()method 
public function send(to_ids:Array, notification:String, email:String = ""):void

Send a notification to a set of users. You can send emails to users that have added the application without any confirmation, or you can direct a user of your application to the URL returned by this function to email users who have not yet added your application.

Parameters
to_ids:Array — Comma-separated list of recipient ids. These must be friends of the logged-in user or people who have added your application.
 
notification:String — FBML for the notifications page.
 
email:String (default = "") — (Optional) FBML for the email. If not passed, no email will be sent.

See also

http://developers.facebook.com/documentation.php?v=1.0&doc=notifications
sendRequest()method 
public function sendRequest(to_ids:Array, type:String, content:String, image:String, invite:Boolean = false):void

Send a request or invitation to a set of users. You can send requests to users that have added the application without any confirmation, or you can direct a user of your application to the URL returned by this function to send requests to users who have not yet added your application.

Parameters
to_ids:Array — Comma-separated list of recipient ids. These must be friends of the logged-in user or people who have added your application.
 
type:String — The type of request/invitation - e.g. the word "event" in "1 event invitation."
 
content:String — Content of the request/invitation. This should be FBML containing only links and the special tag to specify the buttons to be included in the request.
 
image:String — URL of an image to show beside the request. It will be resized to be 100 pixels wide.
 
invite:Boolean (default = false) — Whether to call this an "invitation" or a "request".

See also

http://developers.facebook.com/documentation.php?v=1.0&doc=notifications
