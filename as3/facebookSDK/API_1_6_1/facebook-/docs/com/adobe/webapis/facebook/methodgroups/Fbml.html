Method detail
refreshImgSrc()method
public function refreshImgSrc(url:String):void

Fetches and re-caches the image stored at the given URL, for use in images published to non-canvas pages via the API (e.g. to user profiles via facebook.profile.setFBML, or to news feed via facebook.feed.publishActionOfUser).

Parameters
url:String — Absolute URL from which to refresh the image.

See also

http://developers.facebook.com/documentation.php?v=1.0&method=fbml.refreshImgSrc
refreshRefUrl()method 
public function refreshRefUrl(url:String):void

Fetches and re-caches the content stored at the given URL, for use in a fb:ref FBML tag.

Parameters
url:String — Absolute URL from which to fetch content. This URL should be used in a fb:ref FBML tag.

See also

http://developers.facebook.com/documentation.php?v=1.0&method=fbml.refreshRefUrl
setRefHandle()method 
public function setRefHandle(handle:String, fbml:String):void

Associates a given "handle" with FBML markup so that the handle can be used within the fb:ref FBML tag. A handle is unique within an application and allows an application to publish identical FBML to many user profiles and do subsequent updates without having to republish FBML on behalf of each user.

Parameters
handle:String — Handle to associate with the given FBML.
 
fbml:String — FBML to associate with the given handle.

See also

http://developers.facebook.com/documentation.php?v=1.0&method=fbml.refreshRefUrl
