Classpublic class Profile

Sets the FBML for a user's profile, including the content for both the profile box and the profile actions.

Parameters
markup:String — The FBML intended for the user's profile.
 
uid:int (default = -1) — (Optional) The user whose profile is to be updated. Not allowed for desktop applications (since the application secret is essentially public).
